[파일/디렉토리 명령어]

pwd                	- 현재 작업 디렉토리 출력 (Print Working Directory)
ls                 	- 현재 디렉토리의 파일 및 폴더 목록 출력
ls -al             	- 숨김 파일 포함, 상세 정보와 함께 목록 출력

cd                 	- 홈 디렉토리로 이동 (cd ~와 동일)
cd -               	- 이전 디렉토리로 이동
cd ..              	- 상위 디렉토리로 이동
cd ~               	- 홈 디렉토리로 이동

mkdir 폴더명      	- 새 디렉토리 생성
rm 파일명          	- 파일 삭제

참고: 폴더도 유닉스/리눅스 시스템에서는 "파일"로 간주됨


[vim 에디터 기본 사용법]

vim 파일명         	- 파일 열기 또는 새 파일 생성
i                  	- 입력 모드 진입
esc                	- 명령 모드 진입
:w                 	- 저장
:q                 	- 종료
:wq                	- 저장 후 종료
:q!                	- 저장 없이 강제 종료


[C 컴파일 명령어 (gcc)]

gcc -o 실행파일이름 소스파일.c   - C 파일을 컴파일하여 실행파일 생성
